Project

  • Qiling MCU, Emulation, Python


    Qiling is a cross platform and multi arch lightweight emulator, I joined Qiling group as a core developer. I intruduced MCU emulation module into the emulator, which allows the execution of MCU firmware without real peripheral hardware.

  • Pymx Compiler, RISCV, Python


    Pymx is a compiler written in Python. The compiler can compile a C-like language into rv32im assembly code. It uses LLVM-style IR, I implemented many optimization methods, including global value numbering, dead code eliminate and SSA. The performance of the assembly code generated by the compiler is better than that generated by gcc with O2.

  • RV32-CPU FPGA, Verilog, CPU, RISCV


    The project is a riscv cpu with tomasulo algorithm implemented in Verilog HDL, it works good at 100M on the fpga and it did not show any errors during the experiment. The CPU Supports many useful features, include out-of-order execution, instruction cache, load buffer, etc.
